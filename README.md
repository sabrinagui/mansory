# Mansory - Premium Stone Masonry & Architectural Craftsmanship

A sophisticated, high-end website for a luxury stone masonry contractor specializing in pools, architectural masonry, and hardscape design.

![Mansory Preview](https://images.unsplash.com/photo-1600585154340-be6161a56a0c?w=1200&q=80)

## Features

- **Premium Design**: Elegant, minimalist aesthetic with sophisticated typography
- **Responsive**: Fully responsive with smooth mobile menu
- **Glass Morphism Header**: Fixed header with scroll effects and backdrop blur
- **Modern Layout**: Clean grid-based project showcase
- **Performance**: Optimized HTML & CSS only (no frameworks)
- **Ready for Figma**: Well-organized code structure for easy import

## Tech Stack

- HTML5 (Semantic markup)
- CSS3 (Custom properties, Grid, Flexbox)
- Vanilla JavaScript (Minimal, for header effects & mobile menu)
- Google Fonts (Playfair Display + Outfit)

## Typography

- **Display**: Playfair Display (headlines)
- **Body**: Outfit (text & UI)

## Color Palette

- **Primary**: Charcoal (#1a1a1a) & Black (#0a0a0a)
- **Accent**: Sand (#c4b5a3) & Sand Light (#d9cdbf)
- **Neutrals**: Stone, Cream, Ivory

## Installation

Simply open `index.html` in your browser. No build process required.

```bash
# Clone the repository
git clone https://github.com/sabrinagui/mansory.git

# Open in browser
open index.html
```

## Design Philosophy

This website embodies the principles of high-end architectural design:
- **Timeless elegance** over trends
- **Generous whitespace** for breathing room
- **Premium materials** (stone textures, natural imagery)
- **Sophisticated hierarchy** in typography
- **Subtle animations** that enhance, not distract
